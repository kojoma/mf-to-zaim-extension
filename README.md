# Money Forward to Zaim Syncer

マネーフォワード MEの月次取引履歴画面（`https://moneyforward.com/cf`）から明細データをインテリジェントに抽出し、公式Zaim API（OAuth 1.0a）を使用してZaimへスマートかつ安全にコピー・同期するGoogle Chrome拡張機能です。

---

## 🎨 主な特徴

- **プレミアムなビジュアル**: マネーフォワードのオレンジとZaimのグリーンをブレンドした美しいグラデーションと、すりガラス調（グラスモーフィズム）の高品質なサイドバーUI。
- **自動マッピング & 学習**: マネーフォワードのカテゴリ名をZaimのカテゴリ・ジャンルと自動的に前方・部分一致でマッピング。ユーザーの手動マッピング選択をChromeローカルストレージに記憶する学習機能搭載。
- **口座の完全自動適用**: 設定画面であらかじめ保存したデフォルト口座（支払元・入金先）が同期時に自動で適用されるため、サイドバーが極めてシンプルかつ省スペースに。
- **CORS制限の完全回避**: Web Crypto APIを用いたブラウザネイティブな高速OAuth 1.0a署名（HMAC-SHA1）エンジンを background (サービスワーカー) に実装。
- **安全なデータ管理**: APIキーや認証トークンはブラウザのローカルストレージ（`chrome.storage.local`）にのみ安全に保存され、外部サーバー等には一切送信されません。
- **使いやすい「未選択」スタート**: 解析された明細はデフォルトでチェックが外れた状態（半透明）で一覧表示されるため、大量の明細の中からコピーしたい取引だけを数個チェックして快適に同期可能。
- **「振替」の自動除外**: 口座間の移動（振替）明細は自動的にスキップされ、二重登録を防止。

## 📂 作成ファイル構造

本リポジトリは以下のファイルで構成されています：

- 🛠 **基本構成**
  - `manifest.json` - 拡張機能のメタデータと権限定義 (Manifest V3)
  - `icon.png` - プレミアムデザインのアプリアイコン (生成画像)
- 🧠 **バックグラウンド処理 (署名・通信)**
  - `background.js` - ネイティブOAuth 1.0a署名エンジン & APIプロキシ
- ⚙️ **設定（オプション）画面**
  - `options.html` - 設定画面HTML
  - `options.css` - 美しいグラスモーフィズムデザインのスタイルシート
  - `options.js` - 認証情報の保存・テスト接続・口座一覧の取得、インタラクティブOAuthハンドシェイク
- 📊 **ツールバーポップアップ (クイック確認)**
  - `popup.html` - ツールバーから開く簡易ダッシュボードHTML
  - `popup.js` - 接続ステータスの確認と設定ショートカットのロジック
- 💻 **Money Forward ME 注入画面 (サイドバーUI)**
  - `content.js` - 取引テーブルのDOM解析、カテゴリ自動マッピング、同期ループ、UIアニメーションロジック
  - `content.css` - インジェクションされるプレミアムサイドバーのスタイルシート

## 🚀 動作検証・利用方法

### 1. 拡張機能のロード
1. Google Chrome を開き、URLバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
2. 画面右上にある **「デベロッパー モード」** のトグルスイッチを **ON** にします。
3. 画面左上に出現する **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. 本リポジトリのフォルダを選択します。
5. 拡張機能が正常に読み込まれ、美しいアイコンが表示されればロード完了です！

### 2. Zaim APIの初期設定（アクセストークンの取得）
1. Chromeのツールバーで本拡張機能のアイコンをクリックし、**「API設定画面を開く」** をクリックします。
2. [Zaim Developers](https://dev.zaim.net/home)でアプリを登録して取得した **コンシューマーID (Consumer Key)** と **コンシューマーシークレット (Consumer Secret)** を一番上のフォームに入力します。
   - *※Zaimでアプリ作成時、アクセス権限で「読み込み」および「書き込み」の両方を許可し、ホームページURLとコールバックURLを `http://localhost/` に設定してください。*
3. フォームのすぐ下にある **「アクセストークン取得アシスタント」** カードを使用します：
   - **ステップ 1**: 「Zaim認証ページを開く」ボタンをクリックします。別タブでZaimのアクセス許可画面が開きますので、「許可する」をクリックします。
   - 認可が完了するとブラウザのURLが `http://localhost/?oauth_token=...&oauth_verifier=XXXXXX` にリダイレクトされます（画面に「このサイトにアクセスできません」と表示されますが正常です）。このURL全体をコピーします。
   - **ステップ 2**: コピーしたURLを「ステップ 2」の入力ボックスに貼り付け、**「アクセストークンを取得・適用する」** ボタンをクリックします。
4. 拡張機能が自動的にZaimと通信し、**アクセス トークン** と **アクセス トークン シークレット** を取得し、上のフォームに自動で入力・保存します！
5. 接続テストに成功すると、一番下の「動作環境設定」カードが有効になります。Zaimから自動取得された口座一覧がプルダウンに反映されますので、デフォルトの「支出元口座」と「収入先口座」を選択し、**「デフォルト口座を保存」** をクリックします。

### 3. 同期の実行
1. マネーフォワード MEの収支履歴ページ（`https://moneyforward.com/cf`）を開きます。
2. 画面右下に、美しいグラデーションの丸い **「Zaimへ同期」** ボタンが出現します。
3. ボタンをクリックすると、右端から滑らかに **「Zaim 同期ダッシュボード」** がスライドインします。
4. ページ内の取引明細が自動的に解析され、カード型リストとして一覧表示されます。
   - **デフォルトで未選択状態**: 全明細のチェックボックスは初期状態で「未選択（オフ）」になります。Zaimにコピーしたい明細のみをチェックしてください。
   - 振替（口座間移動）の明細は自動的に除外・スキップされます。
   - 支出（金額がマイナス）と収入（金額がプラス）を自動判別し、色分けされます。
   - Zaimのカテゴリ・ジャンルが自動マッチングされます。
5. 各カード上で、カテゴリを必要に応じて調整します（ここで手動変更したマッピングは自動で記憶されます）。口座は、設定画面で設定したデフォルト口座が自動で適用されます。
6. 最下部の **「選択した明細をZaimに登録」** ボタンをクリックすると、同期処理がスタートし、進捗バーとカードごとのアニメーションで完了状況がリアルタイムに表示されます！
